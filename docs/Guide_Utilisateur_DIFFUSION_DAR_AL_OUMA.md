# Guide utilisateur detaille - DIFFUSION DAR AL OUMA

## 1. Objectif de la plateforme

DIFFUSION DAR AL OUMA est une plateforme de gestion et de partage des livres scolaires. Elle permet de presenter un catalogue moderne, de lire des PDF, d envoyer des demandes de reunion et de commander des livres via WhatsApp.

La plateforme est construite autour de trois profils :

- Super Admin
- Formateur
- Utilisateur

## 2. Navigation generale

Le site est organise autour de plusieurs zones principales :

- Accueil
- Livres
- Details du livre
- Reunions
- Dashboard

La page d accueil donne une vue generale de la plateforme avec :

- une barre de recherche
- les categories
- des livres mis en avant
- des statistiques

## 3. Connexion et creation de compte

Un utilisateur peut creer son compte avec :

- prenom
- nom
- email
- telephone
- mot de passe

Une fois connecte, le systeme dirige automatiquement l utilisateur vers le dashboard qui correspond a son role.

## 4. Espace Super Admin

Le Super Admin est le responsable principal de la plateforme. Il dispose d un acces complet.

Il peut :

- ajouter un livre
- modifier un livre
- supprimer un livre
- importer une image de couverture depuis un fichier
- ajouter ou modifier un PDF
- gerer les comptes des formateurs
- consulter tous les utilisateurs
- confirmer ou refuser les demandes de reunion
- consulter les statistiques
- gerer les categories
- gerer les niveaux

### Gestion des comptes formateurs

Le Super Admin reste le seul responsable de la creation et de l organisation des comptes formateurs.

Il peut :

- creer un formateur
- lui attribuer une matiere
- lui attribuer un niveau
- modifier son role ou sa matiere
- supprimer son compte si necessaire

## 5. Espace Formateur

Le formateur ne voit pas toute la plateforme. Il voit uniquement ce qui concerne sa matiere.

Il peut :

- consulter les livres de sa matiere
- suivre les demandes de reunion associees a sa matiere
- repondre plus facilement aux besoins des utilisateurs

Le formateur ne peut pas :

- acceder aux reglages du Super Admin
- gerer toutes les categories
- gerer tous les comptes

## 6. Espace Utilisateur

L utilisateur final peut etre un eleve, un parent ou une autre personne interessee par les livres scolaires.

Il peut :

- parcourir les livres
- utiliser la recherche
- filtrer les livres par matiere
- filtrer les livres par niveau
- consulter les details complets d un livre
- lire un PDF si disponible
- commander le livre via WhatsApp
- envoyer une demande de reunion

## 7. Fiche detaillee du livre

Chaque fiche contient :

- la couverture du livre
- le titre
- la description
- la matiere
- le niveau
- le bouton Lire PDF
- le bouton Commander via WhatsApp
- le formulaire Demande reunion

### Cas du PDF

Si le PDF existe :

- le bouton Lire PDF ouvre le document dans un nouvel onglet

Si le PDF n existe pas :

- le message "PDF غير متوفر حاليا" apparait

## 8. Demandes de reunion

Les demandes de reunion permettent a l utilisateur d envoyer un message a propos d un livre.

Les statuts possibles sont :

- pending
- confirmed
- refused

Le Super Admin peut :

- confirmer la demande
- refuser la demande

Le formateur peut :

- consulter les demandes reliees a sa matiere

## 9. Commande via WhatsApp

Le bouton de commande ouvre directement WhatsApp avec un message deja prepare.

Avantages :

- gain de temps
- moins d erreurs dans le titre du livre
- contact plus rapide

## 10. Recherche et filtres

La page Livres permet :

- une recherche par mot cle
- un filtre par matiere
- un filtre par niveau
- une lecture rapide des cartes de livres

## 11. Bonnes pratiques

### Pour les livres

- choisir un titre clair
- rediger une description utile
- verifier la matiere
- verifier le niveau
- importer une couverture propre
- verifier le PDF

### Pour les comptes formateurs

- attribuer la bonne matiere
- garder un email valide
- garder le numero de telephone a jour
- limiter les acces sensibles au Super Admin

## 12. Conclusion

DIFFUSION DAR AL OUMA a ete concue pour offrir une experience claire, moderne et professionnelle a tous les profils.

Ce guide peut etre partage avec :

- les administrateurs
- les formateurs
- les eleves
- les parents

Il sert de reference simple pour comprendre rapidement comment utiliser le site.
